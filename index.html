Kong Features

Learn how the most used API Layer can help growing your API business

    Sell your API by creating flexible billing plans and letting the Kong handle the rest.

How does Kong Work?

The Kong sits in front of your API with a processing latency under 10ms

  • Accesible from any client, server, mobile or application

  • Horizontally scalable to handle hundreds of billion of requests

  • Supports any RESTful API, inside or outside your infrastructure
